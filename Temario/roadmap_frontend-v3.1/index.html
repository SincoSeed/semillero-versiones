Sinco Semillero Resumenes de Temas

HTML

HTML (HyperTextMarkupLenguage) es un lenguaje de marcado de páginas web, que sirve para realizar el cuerpo de un diseño de una página web, mediante etiquetas específicas.

CSS

Css (cascade style sheet) es un lenguaje que define los estilos de presentación para los elementos de un documento HTML en un sitio web.

JavaScript

JavaScript es un lenguaje de programación asíncrono, basado en prototipos, que puede ser usado en aplicaciones de cliente como a nivel de servidor gracias a motores que lo interpretan.
