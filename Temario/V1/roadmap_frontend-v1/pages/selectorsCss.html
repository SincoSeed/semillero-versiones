Selectores css

Selector :

Es el nexo entre los elementos (sujeto del selector) de un documento Htlm, XML o SVG y una hoja de estilos

Selectores básicos

Selector Ejemplo Descripción Opciones
.class .table Selecciona todos los elementos que tengan la clase='table'
#id #firtsName Selecciona el elemento con id='firtsName'
elemento th Selecciona todos los elementos < th>
elemento.class secction.table Selecciona todos los elementos < section> con clase='table'
elemento,elemento div,p Selecciona todos los elementos < div> y todos los elementos < p>
elemento elemento secction table Selecciona todos los elementos < table> dentro de los elementos < section>
elemento>elemento div>p Selecciona todos los elementos < p> donde el elemento padre es un < div>
elemento+elemento div+p Selecciona todos los elementos < p> que se colocan inmediatamante después de los elementos < div>
elemento~elemento div~p Selecciona cada elemento < p> precedido del elemento < div>
[ atributo ] [ title ] Trae la coincidencia de elementos en función de la presencia del atributo title
[ atributo=valor ] [ title = 'Pricipal' ] Trae la coincidencia de elementos en función de la presencia del atributo y del valor
[ atributo~=valor ] [ title ~= 'Pricipal' ] Selecciona todos elementos con un atributo titleque contienen la palabra Principal y del valor
[ atributo|=valor ] [ title |= 'Pri' ] Selecciona todos elementos con un atributo title que contienen un valor que comienza en "Pri"
[ atributo^=valor ] a[ href= ^= "https" ] Selecciona cada elemento a cuyo valor de atributo href comienza con "htpps"
[ atributo$=valor ] a[ href= $= ".pdf" ] Selecciona cada elemento a cuyo valor de atributo href termina con ".pdf"
[ atributo*=valor ] a[ href= *= "w3schools" ] Selecciona cada elemento a cuyo valor de atributo href contiene la subcadena"w3schools"

Pseudo-selectores

Selector Ejemplo Descripción Opciones
:active a:active Selecciona el enlace activo
::after p::after Inserta algo después del contenido de cualquier elemento < p>
::before p::before Inserta algo antes del contenido de cualquier elemento < p>
::cheked input:cheked Selecciona cada elemento < input> marcado
::default input::default Selecciona cada elemento < input> predeterminado
:desabled input:desabled Selecciona cada elemento < input> deshabilitados
:empty input:default Selecciona cada elemento < input> predeterminado
