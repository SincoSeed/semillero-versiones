SINTAXIS Y CONSTRUCTORES BÁSICOS

una app o programa es una lista de "instrucciones" para ser "ejecutadas"por una computadora o navegador
en un lenguaje de programación, estas instrucciones de programacion se denominan declaraciones
un programa en JavaScript es una lista de declaraciones de programacion

Las declaraciones de JavaScript se componen de:
valores, operadores, expreciones, palabras clave y comentarios
a lo cual se le llama codigo JavaScript

palabra clave descripcion tips/ejemplos
;

los punto y coma separan las declaraciones de JavaScript y hacen parte de la sintaxis del lenguaje

agregue ; al final de cada instruccion ejecutable

{}

{} se usan tanto en ciclos, como en funciones, ya que en su interior van una o varias lineas de codigo ejecutables, dentro de una misma funcion o ciclo, señaliza de manera ordenada este tipo de procedimientos

al iniciar un ciclo o funcion ayuda al orden colocar

ciclo o funcion {
....contenido....
} cierre del ciclo o funcion

para visualizar de manera jerarquica en caso de tener varios ciclos o funciones anidadas
break

idetifica la interupcion de un proceso de ejecucion ya sea interruptor como en el switch, o un ciclo, como el caso de los, for, while y do while

puede usar el break para parar la ejecuacion, cuando una condicion no se cumpla, o simplemente cuando encuentres lo que necesitasm y asi optimizar recursos
var / const

son las dormas de declaracionde variables usadas en js, hacen practicamente lo mismo, la diferencia es que una es miltinavegador y la otra no

una el const si vas a desarollar para un mavegador como Chrome, Opera o Mozila, si vas a usar desarrollo multinavegador que incluya explores var es la compatible con pracitcamente todos los navegadores

function

palabra reservada para utilizada para declarar funciones

hay varios tipos de fnciones, sin embargo todas se declaran igual, exeptuando la funcion (=>)
return

es usado para retornar algun valor de una funcion, por ejemplo uuna funcion que sume 2 numeros y reotrna el resultado, el return se usaria para retornar un valor, en este caso el resultado de la suma>

usa el return solo cuando sea necesario al 100%, ademas, puede utilizar una variable global, para almacenar el valor que tendria un return, y asi reusar el resultado cada vez que se ejecute la funcion
