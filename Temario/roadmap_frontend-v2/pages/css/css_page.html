Selectores Css

selectores en CSS son etiquetas que permite diseñar, perzonalizar y cambiar elementos de un HTML, aplicando propiedades especificas que cada uno de los selectores posee, para brindar asi un sin fin de posibilidades de diseño.
estos selectores se dividen en selectores y pseudoselectores

los pseudo selectores o pseudo clases, son clases de los elementos HTML que pueden ser tomadas como eventos durante la navegacion, usando las propiedades de el propio elemento, para aplicar un estilo a un momento en especifico dentro de la navegacion de una pagina web
tembien existen los pseudo elementos que basicamente son, elementos que se pueden agregar al html, desde una propiedad de css
